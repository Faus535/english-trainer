# English Fast Learning System

Sistema de aprendizaje rapido de ingles. Todo el contenido esta precargado y organizado por modulos.

## Estructura

| Carpeta | Contenido |
|---------|-----------|
| `english/vocabulary/` | Palabras mas usadas organizadas por frecuencia y categoria |
| `english/verbs/` | Verbos mas usados con conjugaciones y ejemplos |
| `english/grammar/` | Reglas gramaticales esenciales |
| `english/phrases/` | Frases y expresiones cotidianas |
| `english/pronunciation/` | Guia IPA y pronunciacion para hispanohablantes |
| `english/listening/` | Entrenamiento auditivo: formas reducidas, connected speech, ejercicios |
| `english/daily/` | Sistema de habito diario y rutinas para iPhone |
| `english/skills/` | Skills interactivos para practicar |

## Listening (Entrenamiento Auditivo)

### Como suena el ingles real
- `listening/reduced-forms.md` - Formas reducidas: gonna, wanna, shoulda, weak forms, contracciones avanzadas
- `listening/rhythm-and-stress.md` - Ritmo stress-timed, content vs function words, schwa, word stress
- `listening/intonation.md` - Patrones de entonacion, sarcasmo, tag questions, emociones
- `listening/fillers-and-markers.md` - Muletillas (uh, like, you know), discourse markers, tecnica de filtrado
- `listening/accents.md` - American vs British vs Australian vs Irish vs Scottish

### Entrenamiento practico
- `listening/connected-speech-drills.md` - Ejercicios de connected speech (3 niveles + decodificacion)
- `listening/minimal-pairs-drills.md` - Pares minimos: entrenamiento de oido (vocales, consonantes, frases)
- `listening/numbers-spoken.md` - Numeros, fechas, dinero, horas, medidas al oido
- `listening/hardest-phrases.md` - 200 frases mas dificiles de entender para hispanohablantes

### Contenido real
- `listening/slang-modern.md` - Slang actual 2024-2026, abreviaturas de texto
- `listening/training-plan.md` - Plan de entrenamiento: dictation → shadowing → inmersion → conversacion
- `listening/series-movies.md` - Entender series/peliculas: escalera de subtitulos, series por nivel, vocabulario por genero, plan 12 semanas
- `listening/songs.md` - Entender canciones: tecnica de 4 pasos, 30 canciones para entrenar, generos por dificultad

### Estrategia y recursos
- `listening/strategies.md` - Estrategias de listening: top-down vs bottom-up, regla del 70%, no traducir en tiempo real, plan de accion
- `listening/resources.md` - Recursos: podcasts, YouTube, apps, webs gratuitas, rutinas, configuracion del movil para inmersion

> Si lees el 90% pero no entiendes al escuchar: empieza por `strategies.md` (como escuchar), luego `rhythm-and-stress.md` (por que suena raro), `reduced-forms.md` (las trampas), y sigue el `training-plan.md` (el metodo).

## Como empezar

1. Instala Obsidian en tu iPhone (`SETUP-IPHONE.md`)
2. Lee `daily/habit-system.md` y elige tu ancla (cuando vas a estudiar)
3. Empieza con nivel "Minimo" (5 min/dia) y no rompas la racha
4. Sigue el plan semanal de `daily/routines.md`
5. Si lees bien pero no entiendes al escuchar → `listening/reduced-forms.md`
6. Usa los skills para practicar activamente
7. Pide quizzes para evaluar tu progreso
