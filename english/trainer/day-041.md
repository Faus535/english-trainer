# Dia 41 — Repaso th + futuro (30 min)

## Objetivo del dia
Repasar los sonidos /θ/ y /ð/, consolidar will vs going to, y hacer dictation.

## Bloque 1: Repaso de th (10 min)

### Test rapido: /θ/ o /ð/?

Pon la mano en la garganta. Di cada palabra y clasifica:

| Palabra | ¿Vibra? | Sonido |
|---------|---------|--------|
| think | No | /θ/ |
| this | Si | /ð/ |
| three | No | /θ/ |
| the | Si | /ð/ |
| mother | Si | /ð/ |
| both | No | /θ/ |
| weather | Si | /ð/ |
| thank | No | /θ/ |
| together | Si | /ð/ |
| math | No | /θ/ |

**Regla general:** Las palabras gramaticales (the, this, that, these, those, they, them, there, then) usan /ð/. Las palabras de contenido (think, three, thank, math, both) suelen usar /θ/.

### Frases de repaso

Pronuncia cada frase prestando atencion a cada "th":

1. I **think** **this** is **the** right one. → /θ/ /ð/ /ð/
2. **Three** of **them** went **through** **the** door **together**. → /θ/ /ð/ /θ/ /ð/ /ð/
3. My **mother** **thinks** **the** **weather** is nice. → /ð/ /θ/ /ð/ /ð/
4. **Thank** **them** for **both** gifts. → /θ/ /ð/ /θ/

### Repaso will vs going to — autoevaluacion

Completa sin mirar apuntes:

1. (Decision espontanea) Someone knocks → I___ get it! → **'ll**
2. (Plan previo) Manana vuelas a Paris → I___ fly to Paris tomorrow. → **'m going to**
3. (Opinion) Creo que ganaran → I think they ___ win. → **will**
4. (Evidencia) Ves nubes negras → It ___ rain! → **'s going to**
5. (Promesa) Nunca te mentire → I ___ never lie to you. → **will**
6. (Plan) Has comprado los billetes → We ___ travel to Rome. → **'re going to**
