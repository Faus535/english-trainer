# Dia 10 — Pares minimos: /ɒ/ vs /ʌ/ (25 min)

## Objetivo del dia
Distinguir /ɒ/ (hot) de /ʌ/ (hut) y practicar los primeros 20 verbos regulares en pasado.

## Bloque 1: Pares minimos /ɒ/ vs /ʌ/ (10 min)

Otro par problematico para hispanohablantes. Ambos suenan como "a/o" pero son diferentes.

- **/ɒ/** = boca MUY abierta, labios redondeados. Como decir "o" abriendo mucho.
- **/ʌ/** = boca relajada, labios neutros. Un gruñido breve.

| /ɒ/ | /ʌ/ | Significado |
|-----|-----|-------------|
| hot /hɒt/ | hut /hʌt/ | caliente / cabaña |
| cop /kɒp/ | cup /kʌp/ | policia / taza |
| not /nɒt/ | nut /nʌt/ | no / nuez |
| rob /rɒb/ | rub /rʌb/ | robar / frotar |
| got /ɡɒt/ | gut /ɡʌt/ | obtuvo / tripa |
| shot /ʃɒt/ | shut /ʃʌt/ | disparo / cerrar |
| dock /dɒk/ | duck /dʌk/ | muelle / pato |
| body /ˈbɒd.i/ | buddy /ˈbʌd.i/ | cuerpo / amigo |
| lock /lɒk/ | luck /lʌk/ | cerradura / suerte |
| boss /bɒs/ | bus /bʌs/ | jefe / autobus |
| stock /stɒk/ | stuck /stʌk/ | acciones / atascado |
| cot /kɒt/ | cut /kʌt/ | cuna / cortar |

**Practica con frases:**
1. "It's really **hot** in the **hut**." *(Hace mucho calor en la cabaña)*
2. "The **cop** drank from a **cup**." *(El policia bebio de una taza)*
3. "I'm **not** eating **nuts**." *(No estoy comiendo nueces)*
4. "He **got** a **gut** feeling." *(Tuvo una corazonada)*
5. "With good **luck**, you'll **lock** the deal." *(Con buena suerte, cerraras el trato)*

## Bloque 2: Top 100 verbos regulares — primeros 20 (10 min)

Practica estos verbos formando frases en Past Simple.

| Verbo | Pasado | Sonido -ed | Español |
|-------|--------|-----------|---------|
| want | wanted | /ɪd/ | querer |
| use | used | /d/ | usar |
| work | worked | /t/ | trabajar |
| call | called | /d/ | llamar |
| try | tried | /d/ | intentar |
| need | needed | /ɪd/ | necesitar |
| ask | asked | /t/ | preguntar |
| help | helped | /t/ | ayudar |
| talk | talked | /t/ | hablar |
| turn | turned | /d/ | girar |
| start | started | /ɪd/ | empezar |
| show | showed | /d/ | mostrar |
| move | moved | /d/ | mover |
| play | played | /d/ | jugar |
| live | lived | /d/ | vivir |
| believe | believed | /d/ | creer |
| happen | happened | /d/ | ocurrir |
| provide | provided | /ɪd/ | proveer |
| include | included | /ɪd/ | incluir |
| continue | continued | /d/ | continuar |

**Forma 5 frases en pasado:**
1. I **wanted** to go home. *(Quise ir a casa)*
2. She **called** me yesterday. *(Ella me llamo ayer)*
3. We **tried** to help. *(Intentamos ayudar)*
4. He **worked** all day. *(Trabajo todo el dia)*
5. They **started** the project last week. *(Empezaron el proyecto la semana pasada)*
