# Dia 46 — /s/ vs /z/ (price vs prize) (25 min)

## Objetivo del dia
Diferenciar /s/ (sorda) de /z/ (sonora) y practicar condicionales.

## Bloque 1: /s/ vs /z/ (10 min)

En espanol, la /z/ apenas existe. En ingles, MUCHAS palabras que se escriben con "s" se pronuncian /z/.

### Como producir cada sonido

| Sonido | Descripcion | Test |
|--------|-------------|------|
| /s/ | Aire puro, sin vibracion. Como una serpiente: "sssss" | Mano en garganta: NO vibra |
| /z/ | Zumbido, con vibracion. Como una abeja: "zzzzz" | Mano en garganta: SI vibra |

**Truco:** Haz "ssssss" y luego enciende la voz sin mover nada mas. Pasa a "zzzzzz". Siente la diferencia.

### Frases de practica

1. The **price** of the **prize** is high. → /s/ luego /z/
2. Her **eyes** looked like **ice**. → /z/ luego /s/
3. **Please** close your **eyes** in **peace**. → /z/ /z/ /z/ /s/
4. I **advise** you to take my **advice**. → /z/ luego /s/
5. Don't **lose** the **loose** button. → /z/ luego /s/

## Bloque 2: Condicionales — practica mixta (10 min)

### Repaso rapido

| Tipo | Estructura | Uso |
|------|-----------|-----|
| Zero | If + present, present | Verdades generales |
| First | If + present, will + verb | Posibilidad real futura |
| Second | If + past, would + verb | Hipotetico/imaginario |

### Ejercicio: Forma 3 First Conditional + 3 Second Conditional

**First Conditional (posible):**
1. If I study hard, I ___ (pass) the exam. → **will pass**
2. If it ___ (rain), we will cancel the picnic. → **rains**
3. If you ___ (not hurry), you will miss the train. → **don't hurry**

**Second Conditional (hipotetico):**
1. If I ___ (be) rich, I would buy an island. → **were**
2. If she ___ (have) more time, she would learn Japanese. → **had**
3. If we ___ (live) in Australia, we would surf every day. → **lived**

### Ejercicio: ¿First o Second?

1. If I win the lottery, I will buy a house. → **First** (posible, he comprado billete)
2. If I won the lottery, I would buy a house. → **Second** (hipotetico, no voy a ganar)
3. If it rains, I'll take an umbrella. → **First** (posible)
4. If I were you, I would apologize. → **Second** (hipotetico, no soy tu)
