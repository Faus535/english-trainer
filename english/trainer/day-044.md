# Dia 44 — /ʒ/ (vision, pleasure) (25 min)

## Objetivo del dia
Aprender el sonido /ʒ/, uno de los mas raros del ingles, y el Second Conditional.

## Bloque 1: El sonido /ʒ/ (10 min)

/ʒ/ es como /ʃ/ pero **con vibracion de cuerdas vocales**. Es el sonido mas raro del ingles — aparece pocas veces, pero en palabras muy comunes.

### Como producirlo

1. Haz el sonido /ʃ/ ("shhh").
2. Ahora **anade vibracion** — pon la mano en la garganta.
3. El resultado es /ʒ/: un "shhh" que vibra.

### Donde aparece /ʒ/

No hay NINGUNA palabra inglesa que empiece con /ʒ/. Siempre aparece en medio o al final:

| Terminacion | Palabras | IPA |
|-------------|----------|-----|
| -sion | vision /ˈvɪʒ.ən/, decision /dɪˈsɪʒ.ən/, television /ˈtel.ɪ.vɪʒ.ən/ | |
| -sure | pleasure /ˈpleʒ.ər/, measure /ˈmeʒ.ər/, treasure /ˈtreʒ.ər/ | |
| -sual | usual /ˈjuː.ʒu.əl/, casual /ˈkæʒ.u.əl/, visual /ˈvɪʒ.u.əl/ | |
| Otras | garage /ɡəˈrɑːʒ/, beige /beɪʒ/, rouge /ruːʒ/ | |

### /ʃ/ vs /ʒ/ — comparacion

| /ʃ/ (sorda, sin vibracion) | /ʒ/ (sonora, con vibracion) |
|---------------------------|---------------------------|
| pressure /ˈpreʃ.ər/ | pleasure /ˈpleʒ.ər/ |
| nation /ˈneɪ.ʃən/ | television /ˈtel.ɪ.vɪʒ.ən/ |
| sure /ʃʊr/ | measure /ˈmeʒ.ər/ |
| rush /rʌʃ/ | unusual /ʌnˈjuː.ʒu.əl/ |
| ship /ʃɪp/ | vision /ˈvɪʒ.ən/ |
| fashion /ˈfæʃ.ən/ | decision /dɪˈsɪʒ.ən/ |

### Frases de practica

1. It's my **pleasure** to **measure** the **treasure**. → /ʒ/ tres veces
2. I made a **decision** about **television**. → /ʒ/ dos veces
3. This is **usually** a **casual** event. → /ʒ/ dos veces
4. The **garage** is **beige**. → /ʒ/ dos veces
5. Her **vision** is **unusual**. → /ʒ/ dos veces

## Bloque 2: Second Conditional (10 min)

### Estructura

> **If + past simple, would + verbo base**

Se usa para situaciones **hipoteticas** o **imaginarias** — cosas que probablemente NO pasaran.

### Ejemplos

| Situacion hipotetica | Oracion |
|---------------------|---------|
| No tengo mucho dinero, pero imagino... | If I **had** more money, I **would** travel the world. |
| No puedo volar, pero imagino... | If I **could** fly, I **would** visit every country. |
| No hablo ingles perfecto, pero... | If I **spoke** perfect English, I **would** get a better job. |
| No he ganado la loteria, pero... | If I **won** the lottery, I **would** buy a house. |
| No soy presidente, pero... | If I **were** president, I **would** change the law. |

**OJO:** Despues de "if" va **past simple**, NO "would". "If I would have..." es INCORRECTO.

**Nota especial:** Con "to be" se usa **were** para todos los sujetos: If I **were**, If he **were**, If she **were**.

### Ejercicio: Completa

1. If I ___ (have) a million dollars, I ___ (buy) a yacht. → **had**, **would buy**
2. If she ___ (be) here, she ___ (help) us. → **were**, **would help**
3. If we ___ (live) in London, we ___ (speak) English every day. → **lived**, **would speak**
4. If he ___ (can) cook, he ___ (make) dinner. → **could**, **would make**
5. If they ___ (not work) so much, they ___ (have) more free time. → **didn't work**, **would have**
