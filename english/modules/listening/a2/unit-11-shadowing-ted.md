# Unit 11: Shadowing ‚ÄĒ TED Talk Style

## Objetivo

Practicar shadowing con frases mas largas y complejas (8-12 palabras) tomadas del estilo de presentaciones TED Talk. Estas frases son articuladas pero a velocidad natural, con vocabulario mas sofisticado que la conversacion cotidiana. Es el paso intermedio entre el shadowing basico (Unit 7) y el habla rapida informal.

## Contenido

### Diferencia con el shadowing basico

| Shadowing basico (Unit 7) | Shadowing TED Talk |
|---|---|
| Frases de 4-8 palabras | Frases de 8-12 palabras |
| Vocabulario cotidiano | Vocabulario academico/divulgativo |
| Ritmo lento ‚Üí natural | Ritmo natural articulado |
| Contracciones frecuentes | Menos contracciones, mas formas completas |
| Entonacion conversacional | Entonacion de presentacion (mas dramatica) |

### Caracteristicas del habla tipo TED

1. **Articulacion clara pero natural**: los presentadores hablan bien, pero no de forma artificial.
2. **Pausas retoricasas**: hacen pausas antes de puntos importantes para crear efecto.
3. **Variacion de velocidad**: aceleran en informacion secundaria y desaceleran en puntos clave.
4. **Estres enfatico**: ponen estres extra en palabras clave para persuadir.
5. **Entonacion amplia**: usan mas rango tonal que en conversacion normal.

### Metodo para frases largas

Para frases de 8+ palabras, anade un paso extra:

1. **Escucha completa**: entiende toda la frase.
2. **Divide en chunks**: separa la frase en 2-3 grupos de sentido.
3. **Shadowing por chunks**: practica cada chunk por separado.
4. **Shadowing completo**: une los chunks.
5. **Shadowing fluido**: repite hasta que fluya sin esfuerzo.

### Ejercicio 2: Shadowing progresivo

Practica las frases 1, 4 y 7 con el metodo de 5 pasos. Para cada una:
- Primero solo los chunks individuales (3x cada uno)
- Luego la frase completa (5x)

### Ejercicio 3: Estres enfatico

Lee la frase 7 ("People don't buy what you do...") de tres formas:
1. Estres normal
2. Enfatizando "WHY" (como persuadiendo)
3. Enfatizando "BELIEVE" (como concluyendo)

Observa como cambia la sensacion del mensaje.

### Ejercicio 4: Grabacion comparativa

## Repaso

1. Para frases largas, **divide en chunks** de 3-5 palabras antes de hacer shadowing completo
2. En habla tipo TED, las **pausas retoricasas** son tan importantes como las palabras ‚ÄĒ no las ignores
3. La **entonacion de presentacion** es mas amplia que la conversacional ‚ÄĒ exagera los altibajos
4. Las **palabras de contenido** llevan el estres; las palabras de funcion siguen reducidas incluso en habla articulada
5. La **repeticion** es clave: minimo 5 veces cada frase completa, mas si es larga
