# Unit 8: Sentence Stress — El ritmo del ingles

## Objetivo

Comprender que el ingles es un idioma "stress-timed" (basado en acentos), no "syllable-timed" (basado en silabas) como el espanol. Esto significa que las frases se organizan alrededor de las palabras acentuadas, y las no acentuadas se comprimen para caber entre ellas. Por eso "DOGS CHASE CATS" y "The DOGS have been CHASING all the CATS" se dicen en aproximadamente el mismo tiempo.

## Contenido

### Stress-timed vs Syllable-timed

| Espanol (syllable-timed) | Ingles (stress-timed) |
|---|---|
| Cada silaba dura mas o menos lo mismo | Solo las silabas acentuadas tienen duracion regular |
| 10 silabas = el doble de tiempo que 5 | 3 acentos = mismo tiempo sin importar cuantas silabas hay entre ellos |
| El ritmo es "ta-ta-ta-ta-ta" | El ritmo es "ta-TUM-ta-ta-TUM-ta-TUM" |

### La regla fundamental

**Las palabras de contenido se acentuan. Las palabras de funcion se reducen.**

| Palabras de contenido (ACENTUADAS) | Palabras de funcion (reducidas) |
|---|---|
| Sustantivos: DOG, HOUSE, TIME | Articulos: a, an, the |
| Verbos principales: RUN, EAT, THINK | Preposiciones: to, for, at, in, of |
| Adjetivos: BIG, HAPPY, COLD | Pronombres: he, she, it, you, we |
| Adverbios: QUICKLY, NEVER, ALWAYS | Auxiliares: is, are, was, have, can |
| Negativos: NOT, NEVER, DON'T | Conjunciones: and, but, or, that |
| Wh-words: WHO, WHAT, WHERE | Posesivos: my, your, his, her |

### Demostracion: 10 pares de frases

Cada par tiene el mismo numero de acentos (beats) pero diferente numero de silabas. En ingles natural, ambas frases de cada par se dicen en **aproximadamente el mismo tiempo**.

---

#### Par 1 (2 beats)

**A:** "DOGS CHASE CATS."
- 3 silabas, 3 acentos
- /ˈdɒɡz ˈtʃeɪs ˈkæts/

**B:** "The DOGS have been CHASING all the CATS."
- 9 silabas, 3 acentos
- /ðə ˈdɒɡz əv bɪn ˈtʃeɪsɪŋ ɔːl ðə ˈkæts/

**Mismo tiempo.** Las 6 silabas extra (the, have, been, -ing, all, the) se comprimen entre los acentos.

---

### Que pasa entre los acentos

Las silabas no acentuadas se comprimen de varias formas que ya has estudiado:

| Tecnica | Ejemplo | Resultado |
|---|---|---|
| Schwa | "to" → /tə/, "the" → /ðə/ | Vocales se neutralizan |
| Formas debiles | "have" → /əv/, "can" → /kən/ | Auxiliares se reducen |
| Elision | "have been" → /əv bɪn/ o /əbɪn/ | Sonidos desaparecen |
| Linking | "all the" → /ɔːlðə/ | Palabras se conectan |
| Fusion | "going to" → /ˈɡʌnə/ | Palabras se fusionan |

### El error del hispanohablante

El hispanohablante tiende a dar el mismo tiempo y energia a cada silaba. Esto produce un "acento espanol" inmediatamente reconocible. Compara:

**Acento espanol:**
"The-dogs-have-been-cha-sing-all-the-cats" (9 silabas iguales)

**Ingles natural:**
"thuh-DOGS-'v-bin-CHASE-ing-all-thuh-CATS" (3 beats dominantes)

La clave es **exagerar**: haz las acentuadas MAS largas y fuertes, y las no acentuadas MAS cortas y debiles. Al principio sonara exagerado, pero eso es lo correcto.

## Ejercicios

### Ejercicio 1: Cuenta los beats

Cuantas palabras acentuadas (beats) tiene cada frase?

1. "I NEED to TALK to you." → ___ beats
2. "She's been WORKING all DAY." → ___ beats
3. "We should have LEFT EARLIER." → ___ beats
4. "The CHILDREN are PLAYING in the GARDEN." → ___ beats
5. "CAN you HELP me with THIS?" → ___ beats

<details>
<summary>Respuestas</summary>

1. 2 beats (NEED, TALK)
2. 2 beats (WORKING, DAY)
3. 2 beats (LEFT, EARLIER)
4. 3 beats (CHILDREN, PLAYING, GARDEN)
5. 3 beats (CAN — acentuado por ser pregunta, HELP, THIS)
</details>

### Ejercicio 2: Pares de ritmo

Lee cada par en voz alta intentando que duren lo mismo.

| Version corta | Version larga | Beats |
|---|---|---|
| "STOP NOW." | "You should STOP right NOW." | 2 |
| "EAT FISH." | "We should EAT some FISH." | 2 |
| "TRAINS LEAVE LATE." | "The TRAINS have been LEAVING very LATE." | 3 |
| "WRITE MORE." | "You should WRITE some MORE." | 2 |
| "SHE SINGS WELL." | "SHE has been SINGING very WELL." | 3 |

### Ejercicio 3: Marca el estres

Lee cada frase y subraya las palabras acentuadas. Luego lee en voz alta.

1. "I've been waiting for you since three o'clock."
2. "He doesn't know what he's talking about."
3. "Can you pick me up from the airport tomorrow?"
4. "She would have called you if she had known."
5. "The weather has been getting worse all week."

<details>
<summary>Respuestas</summary>

1. "I've been **WAITING** for you since **THREE** o'**CLOCK**."
2. "He **DOESN'T** **KNOW** what he's **TALKING** about."
3. "Can you **PICK** me up from the **AIRPORT** to**MORROW**?"
4. "She would have **CALLED** you if she had **KNOWN**."
5. "The **WEATHER** has been getting **WORSE** all **WEEK**."
</details>

### Ejercicio 4: Metronomos humano

Golpea la mesa (o da palmas) SOLO en las silabas acentuadas mientras lees:

1. [golpe] DOGS [golpe] CHASE [golpe] CATS
2. the [golpe] DOGS have been [golpe] CHASING all the [golpe] CATS
3. [golpe] I've been [golpe] WORKING since [golpe] MONDAY
4. [golpe] SHE could have [golpe] TOLD me [golpe] EARLIER

Los golpes deben estar espaciados de forma regular, como un metronomo. Las silabas sin golpe se "meten" entre los beats.

## Repaso

1. El ingles es **stress-timed**: el ritmo se basa en los acentos, no en las silabas
2. **Palabras de contenido** (sustantivos, verbos, adjetivos, adverbios) se acentuan
3. **Palabras de funcion** (articulos, preposiciones, auxiliares, pronombres) se reducen
4. "DOGS CHASE CATS" y "The DOGS have been CHASING all the CATS" duran aproximadamente lo mismo
5. El error tipico del hispanohablante es dar el mismo tiempo a cada silaba — hay que **exagerar** los contrastes
