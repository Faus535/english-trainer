# Linking Basico

## Objetivo

Entender por que las palabras en ingles se conectan entre si y suenan como una sola. Los nativos NO hacen pausas entre palabras. "Turn it off" suena como "tur-ni-toff". Aprender los 3 tipos de linking basico para dejar de escuchar "ruido" y empezar a escuchar palabras.

## Contenido

### Por que el linking es un problema para hispanohablantes

En espanol, tendemos a separar cada palabra claramente: "voy a ir a la tienda" — cada palabra es distinguible. En ingles, las palabras se fusionan en cadenas de sonido continuas. "I need it" no suena como tres palabras separadas, suena como "ai-nee-dit".

El linking no es hablar rapido ni informal. Es la pronunciacion **normal** y **correcta** del ingles.

### Tipo 1: Consonante + Vocal (C+V Linking)

Cuando una palabra termina en consonante y la siguiente empieza con vocal, la consonante **salta** a la siguiente palabra.

Es como si las palabras se re-silabificaran:

**turn it off** → tur-**ni**-**toff** /ˈtɜːr.nɪ.tɔːf/

La "n" de "turn" se une a "it", y la "t" de "it" se une a "off".

| Frase escrita | Como se re-silabifica | IPA |
|--------------|----------------------|-----|
| turn it off | tur-ni-toff | /ˈtɜːr.nɪ.tɔːf/ |
| pick it up | pi-ki-tup | /ˈpɪ.kɪ.tʌp/ |
| not at all | no-ta-tall | /ˈnɑː.tə.tɔːl/ |
| look at it | loo-ka-tit | /ˈlʊ.kə.tɪt/ |
| get up | ge-tup | /ˈɡɛ.tʌp/ |
| come in | co-min | /ˈkʌ.mɪn/ |
| sit on | si-ton | /ˈsɪ.tɑːn/ |
| think about | thin-ka-bout | /ˈθɪŋ.kə.baʊt/ |
| an apple | a-napple | /ə.ˈnæ.pəl/ |
| some of it | so-mo-vit | /ˈsʌ.mə.vɪt/ |
| run away | ru-na-way | /ˈrʌ.nə.weɪ/ |
| put it on | pu-ti-ton | /ˈpʊ.tɪ.tɑːn/ |
| check it out | che-ki-tout | /ˈtʃɛ.kɪ.taʊt/ |
| made of | may-dov | /ˈmeɪ.dəv/ |
| kind of | kine-dov | /ˈkaɪn.dəv/ |

#### Patron con flap T

Cuando la "t" enlaza con una vocal, en ingles americano se convierte en **flap T** /ɾ/ (suena como una "r" suave espanola o una "d" suave):

| Frase | Con flap T suena como | IPA |
|-------|----------------------|-----|
| put it on | pu-**ri**-ton | /ˈpʊ.ɾɪ.tɑːn/ |
| get out | ge-**rout** | /ˈɡɛ.ɾaʊt/ |
| what is it | wha-**ri**-zit | /ˈwɑː.ɾɪ.zɪt/ |
| shut up | shu-**rup** | /ˈʃʌ.ɾʌp/ |
| a lot of | a-lo-**rov** | /ə.ˈlɑː.ɾəv/ |

### Tipo 2: Vocal + Vocal (V+V Linking)

Cuando una palabra termina en vocal y la siguiente empieza con vocal, se inserta un sonido de conexion:

**Regla A**: Si la primera vocal es /iː/, /ɪ/, /eɪ/, /aɪ/ o /ɔɪ/ → se inserta /j/ (como "y")

| Frase | Linking | IPA |
|-------|---------|-----|
| he asked | he**y**asked | /hiː **j** ɑːskt/ |
| she always | she**y**always | /ʃiː **j** ˈɔːlweɪz/ |
| we eat | we**y**eat | /wiː **j** iːt/ |
| I agree | I**y**agree | /aɪ **j** əˈɡriː/ |
| they are | they**y**are | /ðeɪ **j** ɑːr/ |

**Regla B**: Si la primera vocal es /uː/, /ʊ/, /oʊ/ o /aʊ/ → se inserta /w/

| Frase | Linking | IPA |
|-------|---------|-----|
| go away | go**w**away | /ɡoʊ **w** əˈweɪ/ |
| do it | do**w**it | /duː **w** ɪt/ |
| you are | you**w**are | /juː **w** ɑːr/ |
| two eggs | two**w**eggs | /tuː **w** ɛɡz/ |
| how is | how**w**is | /haʊ **w** ɪz/ |

**Regla C**: Si la primera palabra termina en "r" o "re" → la R se pronuncia (linking R / intrusive R)

| Frase | Linking | IPA |
|-------|---------|-----|
| far away | fa**r**away | /fɑː**r** əˈweɪ/ |
| here is | hee**r**is | /hɪə**r** ɪz/ |
| four o'clock | fou**r**o'clock | /fɔː**r** əˈklɑːk/ |
| where are | whee**r**are | /wɛə**r** ɑːr/ |
| there is | thee**r**is | /ðɛə**r** ɪz/ |

### Tipo 3: Consonante + Consonante (C+C Linking)

Cuando una palabra termina en la misma consonante (o una muy similar) con la que empieza la siguiente, las dos consonantes se fusionan en una sola:

| Frase | Suena como | IPA | Nota |
|-------|-----------|-----|------|
| black coat | bla-coat | /ˈblækoʊt/ | Las dos K se fusionan |
| bus stop | bu-stop | /ˈbʌstɑːp/ | Las dos S se fusionan |
| good day | goo-day | /ˈɡʊdeɪ/ | Las dos D se fusionan |
| hot tea | ho-tea | /ˈhɑːtiː/ | Las dos T se fusionan |
| big game | bi-game | /ˈbɪɡeɪm/ | Las dos G se fusionan |
| red door | re-door | /ˈrɛdɔːr/ | Las dos D se fusionan |
| some money | sə-money | /ˈsʌmʌni/ | Las dos M se fusionan |
| with them | wi-them | /ˈwɪðɛm/ | Los dos TH se fusionan |

Tambien ocurre cuando las consonantes son **similares** (mismo punto de articulacion):

| Frase | Suena como | Nota |
|-------|-----------|------|
| bad day | ba-day | /d/ + /d/ |
| right time | righ-time | /t/ + /t/ |
| help Peter | hel-Peter | /p/ + /p/ |
| big car | bi-car | /ɡ/ + /k/ (mismo punto) |
| good time | goo-time | /d/ + /t/ (mismo punto) |

### Resumen visual de los 3 tipos

```
Tipo 1: C + V → Consonante salta a la vocal
         turn_it_off → tur-ni-toff

Tipo 2: V + V → Se inserta /j/ o /w/
         go_away → gow-away
         she_always → shey-always

Tipo 3: C + C → Consonantes iguales se fusionan
         black_coat → bla-coat
```

## Ejercicios

Para cada par de palabras, di que tipo de linking ocurre (C+V, V+V, o C+C):

2. two hours → ______ (V+V: ow + a, inserta /w/)
3. hot day → ______ (C+C: t + d, similar)
4. come over → ______ (C+V: m + o)
5. she asked → ______ (V+V: ee + a, inserta /j/)

7. pick up → ______ (C+V: k + u)
8. go out → ______ (V+V: ow + ou, inserta /w/)
9. big game → ______ (C+C: g + g, igual)
10. far away → ______ (V+V: linking R)

Escribe como se re-silabifican estas frases con linking:

### Ejercicio 3: Escucha y marca los links

Escucha estas frases con TTS. Dibuja guiones bajos (_) donde escuches linking:

1. Turn it off and put it away.
   - Respuesta: Turn_it_off_and put_it_away
2. I need an apple and an orange.
   - Respuesta: I need_an_apple_and_an_orange

   - Respuesta: Look_at_it from_here

   - Respuesta: She_is_always_at the_office

### Ejercicio 4: Practica oral

Di estas frases conectando las palabras. No hagas pausas entre palabras:

1. Pick it up. (pi-ki-tup)
2. An old apple. (a-nol-dapple)
3. Go away. (gow-away)
4. Not at all. (no-ta-tall)
5. He is a good doctor. (hee-yi-za-good-doctor)
6. Far away from here. (far-away-from-here)
7. She asked us to come in. (she-yasked-us-to-co-min)
8. I need a lot of time. (I-nee-da-lo-dov-time)

## Repaso

Usa estos 5 items como calentamiento en sesiones futuras:

1. **C+V linking**: La consonante final salta a la vocal siguiente ("turn_it_off" → "tur-ni-toff")
2. **V+V linking**: Se inserta /j/ (despues de ee/ay/ai) o /w/ (despues de oo/oh/ow)
3. **C+C linking**: Consonantes iguales o similares se fusionan en una ("black_coat" → "bla-coat")
4. **Flap T**: En ingles americano, T entre vocales suena como R suave ("put it on" → "pu-ri-ton")
5. **Practica rapida**: Di "pick it up and put it on" como una cadena fluida sin pausas
