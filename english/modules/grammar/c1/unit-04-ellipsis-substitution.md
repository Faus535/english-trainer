# Ellipsis and Substitution — Natural Omissions

## Objetivo

Omitir palabras innecesarias como un nativo. Entender y producir las omisiones que hacen que el ingles hablado y escrito suene natural.

## Activacion rapida

| Tipo | Completo | Natural (con ellipsis) |
|------|---------|----------------------|
| Subject + aux | Do you want to come? | **Want to come?** |
| Subject + be | That sounds good. | **Sounds good.** |
| It + be | Is that OK? | **OK?** / **That OK?** |
| Infinitive after want/like/try | I'd love to come. I'd love **to**. | (omitir verbo, mantener "to") |
| If so / if not | If that is so... | **If so**, I'll come. / **If not**, don't worry. |
| So + I think/hope/believe | I think so. / I hope so. | **"Will it rain?" "I hope not."** |
| One/ones (sustitucion) | I like the blue **one**. | (sustituye "shirt/car/etc.") |
| Do so (sustitucion) | If you want to apply, please **do so** before Friday. | (sustituye el verbo anterior) |
| Auxiliary repetition | "Have you been there?" "Yes, I **have**." | (omitir verbo principal) |

**Ellipsis en respuestas cortas:**
- "Are you coming?" → "Might (do)." / "Probably." / "Depends."
- "Can you help?" → "Sure." / "Of course." / "Afraid not."
- "Did you like it?" → "Not really." / "Kind of." / "Sort of."

**Substitution con "so":**
- "Is she coming?" → "I think **so**." / "I don't think **so**." / "I hope **so**."
- "Will it rain?" → "I hope **not**." (NO: ~~I don't hope so~~)

**Error hispanohablante:** Ser demasiado completo. Decir "Yes, I would like to go to the party" cuando un nativo diria "Yeah, I'd love to." Tambien: "I don't hope so" (incorrecto) en vez de "I hope not."

## Ejercicios de produccion

### A. Haz estas respuestas mas naturales (usa ellipsis)

1. "Do you want to go to the cinema?" → "Yes, I would love to go to the cinema."
2. "Is everything OK?" → "Yes, everything is fine."
3. "Have you finished?" → "Yes, I have finished."
4. "Can I borrow your pen?" → "Yes, of course you can borrow my pen."
5. "Are you going to the party?" → "I'm not sure if I'm going to the party. It depends on my schedule."

### B. Completa con so, not, one, ones, do so, o to

6. "Will they win?" — "I hope ___."
7. "Is the meeting cancelled?" — "I don't think ___."
8. "I like this shirt, but I prefer the red ___."
9. "Would you like to join us for dinner?" — "I'd love ___."
10. "If you wish to make a complaint, please ___ in writing."

### C. Traduce al ingles de forma natural (no demasiado completo)

11. "Quieres venir?" — "Claro!"
12. "Ha llamado?" — "Creo que si."
13. "Te gusto la pelicula?" — "No mucho."
14. "Puedes ayudarme?" — "Me temo que no."
15. "Va a llover?" — "Espero que no."

### D. Elige la respuesta mas natural

16. "Did you enjoy the concert?"
    a) "Yes, I enjoyed the concert very much."
    b) "Yeah, I really did."
    c) "Yes, I enjoyed."

17. "Are you tired?"
    a) "A bit."
    b) "Yes, I am tired."
    c) "Yes, I'm a bit tired, actually."

18. "Do you think she'll pass?"
    a) "I think so."
    b) "I think yes."
    c) "I think she will pass."

---

## Respuestas

**A.**
1. "Yeah, I'd love to." / "Sure, sounds good."
2. "Yeah, fine." / "All good."
3. "Yes, I have." / "Yeah, just now."
4. "Sure." / "Of course." / "Go ahead."
5. "Not sure. Depends." / "Maybe. Depends on my schedule."

**B.**
6. so (I hope so.)
7. so (I don't think so.)
8. one (the red one)
9. to (I'd love to.)
10. do so (please do so in writing)

**C.**
11. "Want to come?" — "Sure!" / "Of course!"
12. "Has she called?" — "I think so."
13. "Did you like the film?" — "Not really." / "Not much."
14. "Can you help me?" — "Afraid not." / "I'm afraid I can't."
15. "Is it going to rain?" — "I hope not."

**D.**
16. b) "Yeah, I really did." (a = demasiado formal/completo; c = error gramatical, falta objeto)
17. a) o c) son naturales (a = informal, c = conversacional). b = suena robotico.
18. a) "I think so." (b = error — "think yes" no es ingles; c = correcta pero innecesariamente completa)

## Repaso

1. "Is she coming?" → Respuesta natural (esperas que si): ___
2. "Do you want some coffee?" → Respuesta natural (si): ___
3. Corrige: "Will it work?" — "I don't hope so."
4. Cual es mas natural? a) "Yes, I have done it." b) "Yes, I have."
5. Completa: "Which jacket do you want, the black ___ or the brown ___?"

**Repaso — Respuestas:**
1. "I hope so." / "I think so."
