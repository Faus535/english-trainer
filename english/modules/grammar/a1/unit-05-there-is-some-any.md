# There is/are + Some/Any/No — Activation

## Objetivo

Describir lugares, cantidades y existencia con fluidez. Automatizar there is/are y la eleccion entre some, any y no.

## Activacion rapida

| Estructura | Uso | Ejemplo |
|-----------|-----|---------|
| There is + singular/incontable | Afirmativa | There is a park near my house. / There is some milk. |
| There are + plural | Afirmativa | There are three bedrooms. |
| There isn't / There aren't | Negativa | There isn't any coffee. / There aren't any shops. |
| Is there...? / Are there...? | Pregunta | Is there a bank nearby? |

| Palabra | Contexto |
|---------|----------|
| **some** | Afirmativa, ofrecimientos (Would you like some?) |
| **any** | Negativa, preguntas |
| **no** | Afirmativa con significado negativo (There is **no** milk = There isn't any milk) |

## Ejercicios de produccion

### A. Traduce al ingles

1. Hay un supermercado al lado de mi casa.
2. No hay leche en la nevera.
3. Hay algunas personas esperando fuera.
4. Hay algun restaurante cerca de aqui?
5. No hay nada interesante en la tele.

### B. Describe (escribe 3 frases)

6. Describe tu barrio usando "there is/are" (que hay y que no hay).
7. Describe lo que hay en tu nevera ahora mismo.

### C. Corrige los errores

8. There is some people in the garden.
9. Is there any milk? Yes, there is some milk. (esta bien — confirmalo o corrigelo)
10. There aren't no chairs in the room.

---

### Answer Key

**A.**
1. There is a supermarket next to my house.
2. There isn't any milk in the fridge. / There is no milk in the fridge.
3. There are some people waiting outside.
4. Is there a restaurant near here?
5. There is nothing interesting on TV. / There isn't anything interesting on TV.

**C.**
8. There **are** some people in the garden. (people = plural)
9. Correcta. ("Yes, there is some milk" es valido.)
10. There aren't **any** chairs in the room. / There are **no** chairs in the room. (doble negativa no se usa en ingles estandar)

## Repaso

1. Traduce: "Hay demasiado ruido aqui."
2. Traduce: "No hay ningun problema."
3. Pregunta: "Hay wifi en este hotel?"
4. Corrige: "There is many cars on the road."
5. Escribe 2 frases sobre tu oficina/clase usando "there is/are".

### Repaso — Answer Key

1. There is too much noise here.
2. There is no problem. / There isn't any problem.
3. Is there wifi in this hotel?
4. There **are** many cars on the road.
