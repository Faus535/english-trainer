# First Conditional — Activation

## Objetivo

Hablar de posibilidades reales y futuras con fluidez. Automatizar la estructura "If + present, will + verb" sin mezclar tiempos.

## Activacion rapida

**Estructura:** **If** + present simple, S + **will** + verb base

| Ejemplo | Uso |
|---------|-----|
| If it rains, I'll take an umbrella. | Posibilidad real |
| If you study, you'll pass the exam. | Consecuencia probable |
| If she doesn't hurry, she'll be late. | Advertencia |
| I'll help you if you need me. | (el orden puede invertirse) |

**Errores tipicos:**
- ~~If it will rain~~ → If it **rains** (presente en la clausula if, nunca will)
- ~~If I will have time~~ → If I **have** time

Tambien puedes usar: **unless** = if not
- Unless you hurry, you'll miss the bus. = If you don't hurry, you'll miss the bus.

## Ejercicios de produccion

### A. Traduce al ingles

1. Si llueve manana, nos quedaremos en casa.
2. Si no estudias, no aprobaras el examen.
3. Te llamare si llego tarde.
4. Si consigo el trabajo, nos mudaremos a Madrid.
5. A menos que salgas ahora, perderas el tren.

### B. Completa con tu propia idea

6. If I have free time this weekend, I'll...
7. If the weather is good tomorrow, ...
8. If I don't finish this project on time, ...

### C. Transforma usando "unless"

9. If you don't call me, I won't know. → Unless...
10. If she doesn't apologize, I won't forgive her. → Unless...

### D. Corrige los errores

11. If it will snow, we will stay home.
12. If I will have enough money, I'll buy a new car.
13. Unless you won't hurry, you'll be late.

---

### Answer Key

**A.**
1. If it rains tomorrow, we'll stay at home.
2. If you don't study, you won't pass the exam.
3. I'll call you if I'm late.
4. If I get the job, we'll move to Madrid.
5. Unless you leave now, you'll miss the train.

**C.**
9. Unless you call me, I won't know.
10. Unless she apologizes, I won't forgive her.

**D.**
11. If it **snows**, we will stay home.
12. If I **have** enough money, I'll buy a new car.
13. Unless you hurry, you'll be late. (unless ya es negativo, no se anade won't)

## Repaso

1. Traduce: "Si no duermes bien, manana estaras cansado."
2. Escribe 2 frases sobre lo que haras si tienes vacaciones.
3. Transforma a "unless": "If we don't book now, there won't be any seats left."
4. Corrige: "I'll go to the beach if it will be sunny."
5. Completa: "If you ___ (not/eat) breakfast, you ___ (be) hungry by 11."

### Repaso — Answer Key

1. If you don't sleep well, you'll be tired tomorrow.
3. Unless we book now, there won't be any seats left.
4. I'll go to the beach if it **is** sunny.
5. If you **don't eat** breakfast, you **will be** hungry by 11.
