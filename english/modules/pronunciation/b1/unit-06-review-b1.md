## Objetivo

## El concepto

### Resumen de B1

| Unidad | Concepto clave | Resumen en una frase |
|---|---|---|
| 01 | Stress-timed rhythm | Las silabas acentuadas marcan el ritmo; las no acentuadas se comprimen |
| 02 | Intonation patterns | La melodia cambia el significado: ‚Üė afirmacion, ‚Üó pregunta, ‚ÜĎ sorpresa, ‚Üí sarcasmo |
| 03 | Advanced linking | Sonidos intrusivos /r, w, j/ conectan las palabras sin pausas |
| 04 | Flapping | /t/ y /d/ entre vocales = /…ĺ/ (como la "r" de "pero") en ingles americano |
| 05 | Schwa in sentences | Las function words se reducen a schwa /…ô/ en el habla natural |

### Como se combinan

En una sola frase natural, TODOS estos elementos ocurren a la vez:

**"What are you going to do about it?"**

| Elemento | Donde ocurre |
|---|---|
| Ritmo acentual | Solo **WHAT**, **GO**, **DO**, **BOUT** llevan beat |
| Entonacion | ‚Üė baja al final (Wh-question) |
| Linking | "What‚ÄŅare" se funden |
| Flapping | "what are" = /w…í…ĺ.…ôr/, "about it" = /…ôňąba ä…ĺ.…™t/ |
| Schwa | "are" = /…ôr/, "you" = /j…ô/, "to" = /t…ô/ |

**Resultado**: /ňąw…í…ĺ.…ôr.j…ô ňą…°o ä.…™Ňč.t…ô ňąduňź.…ô.ňąba ä…ĺ.…™t‚Üė/

## Errores comunes

| Area | Error tipico | Solucion |
|---|---|---|
| Ritmo | Sigue sonando syllable-timed | Practica con metronomo. Cada beat = 1 segundo. Comprime las silabas debiles. |
| Entonacion | Sube al final de las Wh-questions | Las Wh-questions BAJAN. Graba y escucha. |
| Linking | Pausa entre palabras que deberian enlazarse | Lee frases como una sola palabra larga. |
| Flapping | No usa flap, o lo usa donde no debe | Flap = /t,d/ entre vocales, antes de silaba NO acentuada. |
| Schwa | Pronuncia todas las vocales claramente | Las function words son invisibles. Solo las content words importan. |
