# Unit 03 — Advanced Linking: Intrusive Sounds and Natural Flow

## Objetivo

Dominar las tecnicas avanzadas de enlace entre palabras que hacen que el ingles suene fluido en vez de entrecortado. En A2 vimos el linking basico (consonante + vocal). Ahora trabajaremos con sonidos intrusivos (/r/, /w/, /j/) y el glottal stop.

## El concepto

En el habla natural, los hablantes nativos NUNCA hacen pausas entre palabras dentro de una frase. Las palabras se funden unas con otras. Para lograr esto, el ingles inserta sonidos de enlace que no existen en la escritura.

### 1. Intrusive R /r/

Cuando una palabra termina en vocal /ɑː, ɔː, ɜː, ə/ y la siguiente empieza con vocal, se inserta una /r/ para conectarlas. Esto ocurre especialmente en ingles britanico.

| Escrito | Hablado | IPA |
|---|---|---|
| the idea‿of | the idea**r**‿of | /ðə aɪˈdɪə.**r**əv/ |
| law‿and order | law**r**‿and order | /lɔː.**r**ənd ˈɔːr.dər/ |
| China‿and Japan | China**r**‿and Japan | /ˈtʃaɪ.nə.**r**ənd dʒəˈpæn/ |
| media‿event | media**r**‿event | /ˈmiː.dɪə.**r**ɪˈvent/ |
| drama‿is | drama**r**‿is | /ˈdrɑː.mə.**r**ɪz/ |

### 2. Linking W /w/

Cuando una palabra termina en una vocal redondeada /uː, oʊ, aʊ/ y la siguiente empieza con vocal, se inserta una /w/.

| Escrito | Hablado | IPA |
|---|---|---|
| do‿it | do**w**‿it | /duː.**w**ɪt/ |
| go‿away | go**w**‿away | /ɡoʊ.**w**ə.ˈweɪ/ |
| you‿and me | you**w**‿and me | /juː.**w**ænd miː/ |
| how‿are you | how**w**‿are you | /haʊ.**w**ɑːr juː/ |
| who‿is it | who**w**‿is it | /huː.**w**ɪz ɪt/ |

### 3. Linking Y /j/

Cuando una palabra termina en /iː, eɪ, aɪ, ɔɪ/ y la siguiente empieza con vocal, se inserta una /j/ (como la "y" española).

| Escrito | Hablado | IPA |
|---|---|---|
| the‿end | the**y**‿end | /ðiː.**j**end/ |
| I‿am | I**y**‿am | /aɪ.**j**æm/ |
| she‿is | she**y**‿is | /ʃiː.**j**ɪz/ |
| we‿all | we**y**‿all | /wiː.**j**ɔːl/ |
| they‿are | they**y**‿are | /ðeɪ.**j**ɑːr/ |

### 4. Glottal Stop /ʔ/

A veces, en lugar de enlazar, los hablantes hacen un corte brusco en la garganta (glottal stop). Ocurre especialmente:
- Antes de una vocal acentuada: "the ʔONLY one"
- En ingles britanico informal: "bu**ʔ**er" (butter), "wa**ʔ**er" (water)
- Para enfasis: "That is ʔABsolutely ʔUNacceptable"

| Escrito | Con glottal stop | IPA |
|---|---|---|
| that is | tha**ʔ** is | /ðæ**ʔ** ɪz/ |
| not at all | no**ʔ** at all | /nɒ**ʔ** ət ˈɔːl/ |
| a bit of | a bi**ʔ** of | /ə bɪ**ʔ** əv/ |

## Errores comunes

| Lo que hacen los hispanohablantes | Lo que deberian hacer |
|---|---|
| Ponen pausas entre todas las palabras: "the / end" | Enlazar: "the‿yend" /ðiː.jend/ |
| No insertan sonidos de enlace: "do-it" suena cortado | Insertar /w/: "do‿wit" /duː.wɪt/ |
| Usan un golpe de glotis (ʔ) donde deberia haber enlace | Los glottal stops solo se usan para enfasis, no como habito |
| Pronuncian cada palabra como una unidad independiente | Las frases son un flujo continuo de sonido |
| Exageran la "r" intrusiva haciendola como /r/ española | La /r/ intrusiva es suave, apenas un toque: /ɹ/ |

**Truco**: Piensa que hablar ingles es como cantar: las palabras se funden en una melodia continua. Si puedes poner el dedo entre dos palabras y sentir una pausa, necesitas enlazarlas mejor.
