# Unit 09 — Registro Academico vs Coloquial

## Objetivo

Dominar la diferencia entre registro formal/academico y registro coloquial/informal en ingles. Un hablante C1 puede cambiar de registro segun la situacion: escribe "furthermore" en un ensayo pero dice "also" en una conversacion. Saber CUANDO usar cada registro es tan importante como conocer las palabras. Usar vocabulario academico en una conversacion informal suena pedante; usar slang en un ensayo suena poco profesional.

### Muletillas informales que NUNCA van en texto academico

| Muletilla | Funcion | Ejemplo informal | Por que es informal |
|---|---|---|---|
| like | pausa / enfasis | "It was, like, really hard." | Muletilla vacia, sin contenido |
| basically | simplificar | "Basically, we need more money." | Suena impreciso en academico |
| sort of / kind of | suavizar | "It's sort of a problem." | Demasiado vago para academico |
| whatever | indiferencia | "We can do whatever works." | Falta de precision |
| anyway | transicion | "Anyway, let's move on." | Demasiado casual |
| you know | buscar confirmacion | "It's, you know, complicated." | Muletilla vacia |
| I mean | reformular | "I mean, it's not that bad." | Suena como duda |
| stuff / things | generalizar | "We need to fix stuff." | Demasiado vago |
| gonna / wanna | contracciones extremas | "I'm gonna try harder." | Inaceptable en escrito formal |
| totally / literally | intensificadores | "I literally died." | Uso hiperbolico, no literal |

### Cuando usar cada registro

| Situacion | Registro | Ejemplo |
|---|---|---|
| Ensayo academico | Formal | "Furthermore, the data demonstrates a significant correlation." |
| Email a tu jefe | Semi-formal | "In addition, the data shows a clear trend." |
| Presentacion de trabajo | Semi-formal a formal | "The results indicate a strong relationship." |
| Email a un colega cercano | Neutro | "Also, the numbers show a clear pattern." |
| Conversacion con amigos | Informal | "Plus, the numbers basically show the same thing." |
| WhatsApp / texto | Muy informal | "Also the numbers are like the same lol" |

### Ejemplo comparativo: El mismo parrafo en dos registros

#### Registro academico

"The aforementioned study demonstrates that approximately 70% of participants exhibited a considerable improvement subsequent to the intervention. Furthermore, the data indicates that hitherto unexplored variables may have facilitated the observed outcomes. Nevertheless, it is necessary to ascertain whether these findings can be replicated prior to drawing definitive conclusions."

#### Registro coloquial

"That study I mentioned shows that about 70% of people got way better after the treatment. Also, the data suggests that stuff we hadn't looked at before might have helped with the results. But still, we need to figure out if we can get the same results again before we jump to conclusions."

**Nota:** Ambos dicen EXACTAMENTE lo mismo. La diferencia es solo el registro.

## Ejercicios

### Ejercicio 1: Formal a informal

Reescribe cada frase en registro coloquial (como hablarías con un amigo):

1. "We must endeavour to ascertain the root cause of the issue." → ___
2. "Numerous participants indicated that the programme was insufficient." → ___
3. "The company will commence operations prior to the end of Q3." → ___
4. "Notwithstanding the challenges, considerable progress was achieved." → ___
5. "Pursuant to our discussion, I shall facilitate the necessary arrangements." → ___

<details>
<summary>Respuestas</summary>

1. "We need to try to figure out what's causing the problem."
2. "A lot of people said the program wasn't enough."
3. "The company's going to start up before the end of Q3." / "They're kicking off before Q3 ends."
4. "Even so, we made a lot of progress." / "Still, we got a lot done."
5. "Following our chat, I'll help sort out what needs to happen." / "Like we talked about, I'll get things set up."

</details>

### Ejercicio 2: Informal a formal

Reescribe cada frase en registro academico:

1. "Basically, a lot of people didn't get the results they wanted." → ___
2. "We need to find out what's going on before we do anything." → ___
3. "Also, the thing I said before about the budget — it's still a problem." → ___
4. "Even though it was hard, we got a lot done." → ___
5. "Instead of buying new stuff, we should use what we already have." → ___

<details>
<summary>Respuestas</summary>

1. "A considerable number of participants failed to obtain the desired outcomes."
2. "It is necessary to ascertain the current situation prior to taking action."
3. "Furthermore, the aforementioned budgetary concerns remain a significant issue."
4. "Despite the considerable challenges, substantial progress was achieved."
5. "In lieu of purchasing additional equipment, we should utilize existing resources."

</details>

### Ejercicio 3: Identifica el registro incorrecto

Cada frase tiene un ERROR de registro. Identifica la palabra fuera de lugar y corrigela:

1. (Email formal a cliente) "We will **kick off** the project on Monday pursuant to the agreed timeline." → ___
2. (Conversacion casual) "I need to **ascertain** whether you're coming to the party tonight." → ___
3. (Ensayo academico) "The results were **basically** consistent with previous findings." → ___
4. (Chat con amigo) "**Nevertheless**, I think we should go to the beach." → ___
5. (Informe profesional) "**Loads of** respondents indicated satisfaction with the service." → ___

<details>
<summary>Respuestas</summary>

1. "kick off" → "commence" (demasiado informal para email formal)
2. "ascertain" → "find out" / "figure out" (demasiado formal para conversacion casual)
3. "basically" → eliminar, o "largely" / "broadly" (muletilla informal en texto academico)
4. "Nevertheless" → "Still" / "But" / "Even so" (demasiado formal para chat)
5. "Loads of" → "A considerable number of" / "Numerous" (demasiado informal para informe)

</details>

### Ejercicio 5: Reescribe el email

Este email es demasiado informal para un cliente. Reescribelo en registro formal:

> "Hey,
>
> So basically, we found out that the thing I told you about last week is still a problem. A lot of people are having issues. We're gonna try to fix it before Friday. Instead of buying new stuff, we'll use what we've got. I'll let you know how it goes.
>
> Cheers"

<details>
<summary>Respuestas</summary>

"Dear [Client name],

I am writing to inform you that the aforementioned issue discussed in our previous correspondence remains unresolved. A considerable number of users have reported difficulties. We shall endeavour to rectify the matter prior to Friday. In lieu of procuring additional resources, we intend to utilize existing infrastructure. I will provide an update in due course.

Kind regards"

</details>
