# Unit 09 — False Friends

## Objetivo

Eliminar de una vez los 15 falsos amigos mas peligrosos entre espanol e ingles. Estas palabras se parecen al espanol pero significan algo completamente diferente. Son trampas que te hacen decir exactamente lo contrario de lo que quieres. Despues de esta unidad, no volveras a caer.

## Palabras

| English | IPA | Spanish PARECE | Realmente significa | Lo que querias decir | Trampa/Note |
|---------|-----|---------------|--------------------|--------------------|-------------|
| actually | /ˈæk.tʃu.ə.li/ | actualmente | en realidad, de hecho | currently, nowadays | ERROR CLASICO. "Actually, I disagree" = "En realidad, no estoy de acuerdo". Para "actualmente" usa "currently". |
| sensible | /ˈsen.sə.bəl/ | sensible | sensato, prudente | sensitive | "A sensible decision" = una decision sensata. Si eres "sensible" en espanol, eres "sensitive" en ingles. |
| realize | /ˈriː.ə.laɪz/ | realizar | darse cuenta | carry out, accomplish | "I realized I was wrong" = me di cuenta de que estaba equivocado. Para "realizar un proyecto" usa "carry out". |
| pretend | /prɪˈtend/ | pretender | fingir | intend, aim | "Don't pretend you didn't know" = no finjas que no sabias. Para "pretender" usa "intend" o "aim". |
| eventually | /ɪˈven.tʃu.ə.li/ | eventualmente | finalmente, al final | possibly, perhaps | "He eventually agreed" = al final acepto. Para "eventualmente" (quiza) usa "possibly". |
| assist | /əˈsɪst/ | asistir | ayudar | attend | "I'll assist you" = te ayudare. Para "asistir a una reunion" usa "attend". |
| fabric | /ˈfæb.rɪk/ | fabrica | tela, tejido | factory | "Cotton fabric" = tela de algodon. "Fabrica" = factory. |
| library | /ˈlaɪ.brer.i/ | libreria | biblioteca | bookshop/bookstore | "Go to the library" = ir a la biblioteca. "Libreria" = bookshop. |
| success | /səkˈses/ | suceso | exito | event, incident | "A great success" = un gran exito. "Suceso" = event/incident. |
| embarrassed | /ɪmˈbær.əst/ | embarazada | avergonzado | pregnant | "I'm embarrassed" = estoy avergonzado. "Embarazada" = pregnant. CLASICO error con anecdotas divertidas. |
| sympathetic | /ˌsɪm.pəˈθet.ɪk/ | simpatico | comprensivo, compasivo | nice, friendly, likeable | "She was sympathetic" = fue comprensiva. Para "simpatico" usa "nice" o "friendly". |
| argument | /ˈɑːr.ɡjə.mənt/ | argumento | discusion, pelea verbal | plot (pelicula), point | "We had an argument" = tuvimos una discusion. "Argumento de pelicula" = plot. |
| exit | /ˈeɡ.zɪt/ | exito | salida | success | "Emergency exit" = salida de emergencia. "Exito" = success. |
| record | /rɪˈkɔːrd/ (v) /ˈrek.ɚd/ (n) | recordar | grabar; registro | remember | "Record a video" = grabar un video. "Recordar" = remember. |
| resume | /rɪˈzuːm/ | resumir | reanudar; curriculum | summarize | "Let's resume the meeting" = reanudemos la reunion. "Resumir" = summarize. "Resume" (n) /ˈrez.ə.meɪ/ = CV. |

## Ejercicios

### Choose the correct word

1. I need to (attend / assist) the meeting at 3 PM.
2. She (actually / currently) works at Google.
3. I didn't (realize / carry out) the mistake until later.
4. He's just (pretending / intending) to be sick.
5. (Eventually / Possibly), we found a solution.
6. That was a very (sensible / sensitive) decision.
7. I was so (embarrassed / pregnant) when I tripped in front of everyone.
8. The cotton (fabric / factory) is very soft.
9. Let's go to the (library / bookshop) to borrow some books.
10. His speech was a huge (success / event).

<details>
<summary>Respuestas</summary>

1. attend (asistir a = attend)
2. currently (actualmente = currently)
3. realize (darse cuenta = realize — aqui si funciona)
4. pretending (fingir = pretend)
5. Eventually (finalmente = eventually)
6. sensible (sensato = sensible)
7. embarrassed (avergonzado = embarrassed)
8. fabric (tela = fabric)
9. library (biblioteca = library)
10. success (exito = success)

</details>

### Translate to English (watch out for false friends!)

1. Actualmente trabajo desde casa.
2. No pretendo ser perfecto, solo intento mejorar.
3. Ella es muy sensible — llora con las peliculas.
4. Asisti a una conferencia muy interesante.
5. Al final, recordo donde habia dejado las llaves.

<details>
<summary>Respuestas</summary>

1. I **currently** work from home. (NOT "actually")
2. I don't **intend** to be perfect, I just try to improve. (NOT "pretend")
3. She's very **sensitive** — she cries during movies. (NOT "sensible")
4. I **attended** a very interesting conference. (NOT "assisted")
5. **Eventually**, he/she **remembered** where he/she had left the keys. (NOT "recorded")

</details>

## Repaso

| Front | Back |
|-------|------|
| actually ≠ actualmente | **actually** = en realidad. "Actualmente" = **currently**. |
| sensible ≠ sensible | **sensible** = sensato. "Sensible" (esp) = **sensitive**. |
| pretend ≠ pretender | **pretend** = fingir. "Pretender" = **intend**. |
| embarrassed ≠ embarazada | **embarrassed** = avergonzado. "Embarazada" = **pregnant**. |
| eventually ≠ eventualmente | **eventually** = finalmente. "Eventualmente" = **possibly**. |
